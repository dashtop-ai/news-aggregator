# News Aggregator

AI-powered news aggregator for Dashtop. Curated feed with smart summaries and topic search.

## Features

- **Curated feed** -- 8 priority-ranked stories from your chosen sources
- **Smart summaries** -- Click any headline to expand an AI summary
- **Category filters** -- All, AI & Tech, Dev, Business, Science
- **Save articles** -- Star items to read later
- **Source management** -- Add and toggle news sources
- **Chat search** -- Ask for top stories, summaries, or search by topic

## Install

Install in Dashtop via Settings > Packages, or:

```bash
npm install
npm run pack
# -> dist/dashtop-ai-news-aggregator-v1.0.0.dashtop
```

## Develop

```bash
npm install
npm run dev
```
